# Parameterized function:
```
let x = myFunction(4, 3);
function myFunction(a, b) {
  return a * b;
}
```
# Arrow function
```
let myFunction = (a, b) => a * b;
document.getElementById("demo").innerHTML = myFunction(4, 5);
```
